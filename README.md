# English

## About

Python Engineering Handbook is an open-source collection of production-grade Python development standards and best practices.

The project combines practical guidelines for:

* Clean Code
* PEP 8
* Type Hints
* Async Programming
* FastAPI
* Django / DRF
* Aiogram
* Security
* Architecture Patterns
* Docker
* CI/CD
* DevOps Practices

The goal of the project is to provide a single reference for building maintainable, secure and scalable Python applications.
